# Deloitte RFP Command Center - Enterprise AI Edition

An intelligent, multi-engine platform for Enterprise RFP Analysis and Proposal Generation.

## 🚀 Features
- **Multi-Engine AI Orchestration**: Seamlessly switches between OpenAI (GPT-4o), Groq (Llama 3.3 70B), and Google Gemini.
- **Vercel Unified Deployment**: Single-domain hosting for both React Frontend and Node.js Backend.
- **Direct Document Ingestion**: Supports PDF, Docx, and CSV parsing for automated analysis.
- **Enterprise UI**: Tech-giant aesthetic with dark mode support and interactive dashboards.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Recharts, Lucide Icons.
- **Backend**: Node.js, Express, Prisma ORM.
- **AI**: OpenAI SDK, Groq SDK, Google Generative AI SDK.
- **Deployment**: Vercel (Serverless Functions + Static Hosting).

## 🛠️ Setup
1. Clone the repository.
2. Install dependencies in `backend` and `frontend`.
3. Configure `.env` in `backend` with your API keys.
4. Run `npm run dev` in both directories.
